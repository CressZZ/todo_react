# Todo-List 

## local 실행
```
npm i
npm start
```

## 구현 기능
- 할일 추가 버튼을 눌러서 할 일을 추가한다.
- 최초 추가시 미완료 카테고리에 추가된다.
- 체크박스 클릭시 미완료/완료가 재분류 되어 re-render 된다.
- (-) 삭제 버튼을 눌러 할 일 리스트에서 삭제한다.

## 기술 스택
- React (create-react-app)
- Redux, React-Redux
- sass, node-sass
